Disease Prediction by Registration and Giving the Symptoms
How does the project work?

We use a decision tree Machine Learning model to train the prediction model on a public database on kaggel website, first we use the training data to train the model then we test the accuracy using the test dataset, we ended up with an accuracy of 97 percent for the model. reference

The model is trained on 132 symptoms and 41 diseases, the model takes the symptoms from the user and predicts the disease.
